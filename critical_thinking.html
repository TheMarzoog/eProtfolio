Prime Numbers Linked List Project

In the computer science II course during my freshman year, we were required to implement a linked list that holed only prime numbers. We have to implement four functions. First, A function that takes two arguments the first one is the lowest possible value in the linked list the second argument is the numbers on prime numbers that you want to generate. Second, A function that removes the duplicated values in the linked list. Third, A function that merges four different linked lists. Last, A function that revers the linked list.

Reflection

In this assignment, I learn how to come up with a solution that meets the requirement and improves my solution gradually. I started with the simplest way to generate a prime number using a recursive method which chick if any number smaller a given number is divisible by it or not. If none of the numbers is divisible then It's a prime number. The problem with this approach that I have to check with a lot of numbers that are not divisible which made my program slow and only can generate the first 6000 prime number. To make it faster after 2 I only chick the odd numbers if they are prime or not. Also, I chick with the numbers that are smaller than half of the input this made my program faster and raised the limit to the first 20000 prime numbers. To make it even faster I changed my approach checking only the odd number after two and only trying to divide by odd number which raised the limit to the first 31000 prime numbers. For the second requirement, I implemented the Bubble sort algorithm due to the limitation of the linked list data structure which is having to go throw linear time element access. The last two requirements wore easy to implement. This project taught me how to think critically to find a solution and improve my solution.

Algorithms assignment 2

In the Algorithms course during my junior year, we were given a problem set that contains seven questions. We had to solve this problem and try to improve the solution gradually. Also, we have to analyze the solution in terms of space and time complexity. For this assignment, the doctor gave us the choice of the programming language that we have to use.

Reflection

This assignment was challenging to solve and learned a lot from it I'll high light some of the things a learn. One of the questions was Find in constant space where the circular is starting in a circular linked list such that, the last pointer could point to any node in the list. To solve this problem, I created to pointers to the head node then I moved them with a different speed such that, the first pointer moves one step forward and the other move two steps. when the pointer meets, I reset one of them to the head of the linked list. Finally, I move them again but this time at the same speed just one step at a time they will meet at the starting point of the circular linked list.
